# 麦当劳美食推荐插件 (astrbot_plugin_mcd_recommender)

基于 AstrBot 框架的麦当劳美食推荐插件。检测到用户自定义关键词后，调用大语言模型 (LLM) 结合麦当劳 MCP 实时数据，智能推荐麦当劳美食。

## 功能特性

- 🔑 **自定义触发关键词** - 在后台配置关键词，检测到即触发推荐
- 🤖 **LLM 智能推荐** - 结合用户需求和餐品数据，个性化推荐
- 🍔 **实时餐品数据** - 通过麦当劳 MCP API 获取真实门店菜单
- 🎫 **优惠券信息** - 自动显示可用优惠券
- 📊 **营养参考** - 提供餐品营养信息参考
- 🎉 **活动日历** - 展示当前营销活动
- 🎨 **多种推荐风格** - 活泼可爱/专业简洁/幽默风趣/温暖治愈
- 📍 **城市自动识别** - 自动识别用户提到的城市
- ⌨️ **指令支持** - 支持 `/mcd` 指令手动触发

## 安装方法

### 方法一：通过 AstrBot 管理面板安装

1. 在 AstrBot 管理面板中进入「插件市场」
2. 搜索「麦当劳美食推荐」
3. 点击安装

### 方法二：手动安装

1. 将本插件文件夹复制到 AstrBot 的 `data/plugins/` 目录下
2. 重启 AstrBot 或在管理面板中刷新插件列表
3. 在插件管理中找到「麦当劳美食推荐」并启用

## 配置说明

在 AstrBot 管理面板 -> 插件管理 -> 麦当劳美食推荐 -> 配置中设置：

### 必要配置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 麦当劳 MCP Token | 在 https://open.mcd.cn/mcp/doc 申请 | 空 |

### 触发配置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 触发关键词 | 逗号分隔，检测到这些词时触发推荐 | 吃什么,麦当劳,麦麦,今天吃啥,午饭,晚饭,早餐,夜宵 |
| 关键词匹配模式 | 包含匹配 / 精确匹配 / 正则匹配 | 包含匹配 |

### 门店配置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 默认城市 | 用户未指定城市时使用 | 北京市 |
| 默认门店编码 | 填写后直接使用该门店（可选） | 空 |
| 点餐方式 | 到店取餐 / 麦乐送外送 | 到店取餐 |

### 推荐配置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 推荐风格 | 活泼可爱 / 专业简洁 / 幽默风趣 / 温暖治愈 | 活泼可爱 |
| 显示营养信息 | 是否展示营养信息 | 开启 |
| 显示优惠券信息 | 是否展示优惠券 | 开启 |
| 最大推荐数量 | 单次最多推荐几款 | 5 |
| 使用的 LLM 提供商 | 选择用于生成推荐的模型 | 默认 |

## 使用方法

### 方式一：关键词触发

在聊天中发送包含触发关键词的消息即可，例如：

- "今天午饭吃什么好呢？"
- "推荐一下麦当劳的早餐"
- "上海有什么好吃的麦麦？"
- "我想吃点低卡的，麦当劳有推荐吗？"

### 方式二：指令触发

使用 `/mcd` 指令：

```
/mcd 推荐一个双人套餐
/mcd 减脂期可以吃什么
/mcd 深圳南山附近的店
```

## 申请 MCP Token

1. 访问 https://open.mcd.cn/mcp/doc
2. 点击右上角「登录」，使用手机号验证登录
3. 登录成功后点击「控制台」
4. 点击「激活」按钮，申请 MCP Token
5. 同意服务协议后即可获取 Token
6. 将 Token 填入插件配置中

## 支持的 MCP 工具

本插件使用以下麦当劳 MCP 工具：

- `query-nearby-stores` - 查询附近门店
- `query-meals` - 查询门店可售餐品
- `query-store-coupons` - 查询门店可用优惠券
- `list-nutrition-foods` - 餐品营养信息
- `campaign-calendar` - 活动日历

## 注意事项

1. **MCP Token 安全**：请妥善保管 MCP Token，不要泄露给他人
2. **调用频率**：每个 Token 每分钟最多 600 次请求，请合理使用
3. **数据准确性**：餐品价格和供应情况以实际门店为准
4. **网络要求**：服务器需要能够访问 `https://mcp.mcd.cn`
5. **降级机制**：当 MCP 服务不可用时，插件会使用通用推荐作为降级

## 常见问题

**Q: 为什么检测不到关键词？**
A: 请检查关键词匹配模式设置。包含匹配会匹配消息中的任意位置；精确匹配需要消息完全等于关键词。

**Q: 为什么没有返回实时餐品数据？**
A: 请检查 MCP Token 是否正确配置，以及服务器是否能访问麦当劳 MCP 服务。查看日志获取详细错误信息。

**Q: 可以支持外送点餐吗？**
A: 可以，在配置中将「点餐方式」改为「麦乐送外送」即可。

**Q: 如何添加更多城市支持？**
A: 插件已内置大部分中国主要城市的识别。如果需要支持更多城市，可以在 `main.py` 的 `_extract_city` 方法中添加。
